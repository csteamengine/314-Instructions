Welcome to the our aiplane design page! Here you can find the step by step instructions for building an airplane.

This website is a result of a group project in our English 314 class at Iowa State University. The goal was to create a technical description of how to build and launch a paper airplane. We decided to go above and beyond and create a fully functional active website simply because websites are more interactive and easier to follow than a booklet or pamphlet. We only had to create instructions for one design, but we decided to create three to fill the site up more. One benefit of using a website is that we can use features such as image carousels or collapsible content to make the steps more obvious and easier to follow. We hope you enjoy our site and find in useful, especially if you are our professor!.
